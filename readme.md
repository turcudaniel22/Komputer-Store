### Prerequisites  

- **Node.js** and **npm** (for SASS compilation).  

### Steps  

1. **Clone the Repository**:  
   ```bash
   git clone https://github.com/turcudaniel22/Komputer-Store.git
   cd computer-store
   ```

2. **Open the Project**:  
   Open `html/index.html` in your browser to view the store.
